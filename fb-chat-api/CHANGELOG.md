# Changelog

All notable changes to this project will be documented in this file.

## [2.0.0] - 2025-11-02

### Added
- Initial release of FCA adapted for GoatBot-V2
- Clean login function export compatible with GoatBot-V2
- TypeScript definitions (index.d.ts)
- Comprehensive README with examples
- Example usage file
- Modern async/await support
- Promise-based API alongside callback support

### Changed
- Simplified entry point from original fca-priyansh
- Removed unnecessary global configuration complexity
- Streamlined initialization process
- Updated dependencies to stable versions
- Improved error handling

### Features
- ✅ Full MQTT support via listenMqtt
- ✅ All standard FCA message functions
- ✅ Thread and group management
- ✅ User information retrieval
- ✅ Typing indicators and read receipts
- ✅ Message reactions and editing
- ✅ Compatible with GoatBot-V2 architecture

### Based On
- fca-priyansh v19.0.0
- Inspiration from ws3-fca and nexus-fca GoatBot-V2 adaptations
